🔑 ¿Cómo obtener tu API key?

1Ve a console.anthropic.com y crea una cuenta con tu email.
2Ve a Settings → API Keys → Create Key.
3Copia la key (sk-ant-api03-…) — solo se muestra una vez.
4Ve a Billing → Add Credit y carga mínimo $5 USD.
5Pega la key en la app. ¡Listo!
💰 ¿Cuánto dura el crédito?
Con $5 USD tienes aprox. 400–500 análisis completos (~$0.01 c/u). El crédito no vence.
✅ Google Gemini tiene una capa gratuita — sin necesidad de tarjeta de crédito para empezar.
1Ve a aistudio.google.com e inicia sesión con tu cuenta Google.
2Haz clic en Get API Key → Create API Key (arriba a la izquierda).
3Selecciona un proyecto de Google Cloud (o crea uno nuevo gratis).
4Copia la key generada (AIzaSy…) y pégala en la app.
🎁 ¿Cuánto puedo usar gratis?
Con el plan gratuito tienes 1.500 solicitudes/día con Gemini 1.5 Flash — suficiente para cientos de análisis al día sin pagar nada.
